## 🇺🇸 English Version

### 🔭 The Vision
Clock-Team is a mobile-first web application designed for distributed teams. It eliminates the friction of time-zone math by providing a sleek, cyberpunk-inspired dashboard where you can see your entire team's local time, location, and role at a glance.

### ✨ Key Features
* **Living UI:** Elements breathe and float using physics-based animations powered by Framer Motion.
* **Mobile-First Dashboard:** Highly optimized bottom navigation for quick access to Orbit, Team, Search, and User profiles.
* **Modern Stack:** Leveraging the extreme speed of Vite and the new CSS-first engine of Tailwind v4.

## 🇺🇸 Versión en inglés

### 🔭 La visión
Clock-Team es una aplicación web diseñada para dispositivos móviles y equipos distribuidos. Elimina la complejidad de los cálculos de zonas horarias al proporcionar un panel de control elegante, de inspiración ciberpunk, donde puedes ver de un vistazo la hora local, la ubicación y el rol de todo tu equipo.

### ✨ Características principales

* **Interfaz dinámica:** Los elementos cobran vida y se mueven gracias a animaciones basadas en la física, impulsadas por Framer Motion.

* **Panel de control optimizado para móviles:** Navegación inferior altamente optimizada para un acceso rápido a Orbit, Equipo, Búsqueda y Perfiles de usuario.

* **Tecnología moderna:** Aprovecha la velocidad extrema de Vite y el nuevo motor CSS-first de Tailwind v4.
